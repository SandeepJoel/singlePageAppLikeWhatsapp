# SinglePage Application Like Whatsapp

### How to run this app?

- Clone this repo
- Run `npm i`
- Run `npm run dev`
- Open `http://localhost:3000` in your browser

**Note**: Socket will run in `http://localhost:4000`. So be careful not to run any other service in that port.
